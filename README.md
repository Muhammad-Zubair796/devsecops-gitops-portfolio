# 🚀 Interactive DevSecOps & GitOps Portfolio

[![Live Interactive Demo](https://img.shields.io/badge/View_Live_Interactive_Demo-0f172a?style=for-the-badge&logo=render&logoColor=white)](https://devsecops-gitops-portfolio.onrender.com)

Welcome to my interactive engineering portfolio! This repository contains the frontend source code for a custom-built, interactive dashboard designed to showcase my latest **End-to-End DevSecOps & GitOps Pipeline**.

## 📌 About This Dashboard
Instead of a standard text document, I engineered this interactive web application using **HTML5, Tailwind CSS, and Vanilla JavaScript** to provide a "Post-Mortem" view of my infrastructure. 

It features:
* **Interactive Pipeline Timeline:** Step-by-step visualization of the CI/CD lifecycle.
* **Simulated Terminal:** Real-time typing effects showing the exact Linux, Docker, and Kubernetes commands executed.
* **Engineering Deep-Dives:** Dedicated tabs explaining the Execution Context, Error Logs, and Root Cause Resolutions for complex infrastructure bottlenecks.
* **Codebase Architecture View:** An IDE-like interface explaining the `Jenkinsfile`, `Dockerfile`, and Kubernetes manifests.

## 🛠️ The Pipeline Architecture Showcased
The live demo visualizes a pipeline built with:
* **AWS EC2** (Ubuntu Linux)
* **Jenkins** (Continuous Integration)
* **SonarQube** (Static Application Security Testing)
* **Docker** (Containerization)
* **Kubernetes / Minikube** (Container Orchestration)
* **Argo CD** (Declarative GitOps Continuous Deployment)
